## App Name

MPNMJEC Bus Tracker

## Short Description

Real-time college bus tracking for MPNMJEC students.

## Full Description

MPNMJEC Bus Tracker helps students view live college bus movement, route details, stop information, and active trip status.

Students can search pickup and destination stops, check available buses, and track active buses on a live map. The app is designed for MPNMJEC transport users and depends on active trip data from the college bus tracking system.

Key features:

- View active college buses
- Search pickup and destination stops
- Check available buses for selected routes
- Track bus movement on a live map
- View upcoming stops and route progress
- Use stop arrival alarm preferences stored on the device
